# NirbharID 🚀

Self-Sovereign Identity (SSI) system for migrant workforce mobility using Decentralized Identifiers (DIDs) and Verifiable Credentials.

## 🔍 Problem
Migrant workers face exclusion due to fragmented documentation, lack of portable identity, and unverifiable skill records.

## 💡 Solution
NirbharID enables workers to own and share their identity and credentials securely using SSI principles.

## ✨ Features
- Decentralized Identity (DID)
- Verifiable Credentials (Skill, Identity, Welfare)
- QR-based Verification
- Multilingual Support (English, Hindi, Bengali)
- Credential Status (🟢 Verified, 🟡 Expiring, 🔴 Invalid)

## 🧱 Tech Stack
- React Native (Expo)
- Node.js Backend
- Vite (Verifier App)
